Overview
SmartStudent is a console-based Student Management System developed using Java, JDBC, and MySQL. The application enables administrators to securely manage student records through a simple menu-driven interface. It provides complete CRUD functionality, search capabilities, and database integration, demonstrating practical use of Java database connectivity and object-oriented programming concepts.

Features
Admin Login Authentication
Secure admin login before accessing the system
Prevents unauthorized access to student records
Credential validation using username and password
Student Record Management (CRUD)
Add new student records
View all students stored in the database
Update student information
Delete existing student records
Advanced Search Functionality
Search students by Roll Number
Search students by Department
Search students based on Marks criteria
MySQL Database Integration
Persistent storage using MySQL
JDBC-based database connectivity
Prepared Statements used for secure SQL execution
User-Friendly Console Interface
Menu-driven navigation
Simple and interactive user experience
Easy management of student data
